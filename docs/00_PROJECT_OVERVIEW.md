# New Path Walker Project Overview

New Path Walker (NPW) is a secure, faith-centered community platform for people
experiencing major life transitions. The platform will provide encouragement,
Scripture, personal stories, private discussions, prayer-oriented spaces, and
member-only resources.

NPW is not therapy, counseling, psychiatric treatment, crisis intervention,
medical advice, legal advice, or financial advice. Product language and future
workflows must keep these boundaries clear.

## Step 1 Goal

Step 1 establishes the technical foundation only:

- Next.js App Router
- TypeScript
- Tailwind CSS
- Prisma with MySQL configuration
- Auth.js package foundation
- Nodemailer package foundation
- Placeholder public and protected pages
- Scalable source structure
- Project documentation

No production features are implemented in Step 1.
