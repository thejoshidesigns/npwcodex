# Tech Stack

- Hosting: Hostinger Business
- Language: TypeScript
- Framework: Next.js App Router
- Styling: Tailwind CSS
- Database: MySQL
- ORM: Prisma
- Authentication: Auth.js / NextAuth credentials provider
- Email: Nodemailer with Hostinger SMTP
- Version control: Git
- Package manager: npm

## Notes

GitHub Pages can preview static output, but future authenticated and PHP-free
dynamic features should run on Hostinger or another Node-capable deployment
target. GitHub remains the source of truth for version control.
